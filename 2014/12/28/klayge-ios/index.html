iOS Cross Compile

иҝҷйғЁеҲҶеӯҰд№ дәҶAndroidйғЁеҲҶзҡ„еӨ„зҗҶпјҢдҪҝз”ЁдәҶios-cmakeдҪңдёәCMAKE_TOOLCHAIN_FILEеҸӮж•°дј иҝӣеҺ»пјӣиҝҷдёӘж–Үд»¶йҮҢдё»иҰҒиҝӣиЎҢдәҶд»ҘдёӢи®ҫзҪ®

  • ејәиЎҢи®ҫзҪ®зј–иҜ‘еҷЁдёә/usr/bin/clangпјҢдёҚз„¶зӣҙжҺҘдҪҝз”ЁXcode.appйҮҢзҡ„ClangдјҡжҸҗзӨә: The C compiler вҖң/Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/bin/clangвҖқ is not able to compile a simple test program.пјӣ
  • ејәиЎҢи·іиҝҮзј–иҜ‘еҷЁжЈҖжҹҘпјӣ
  • и®ҫзҪ®иӢҘе№Ізҡ„е®ҸпјҢиӯ¬еҰӮset (IOS True)пјӣ
  • иҮӘеҠЁжҗңзҙўе№¶и®ҫзҪ®CMAKE_OSX_SYSROOTпјҢиҝҷдёӘжҳҜеҢәеҲҶзј–иҜ‘еҮәOSXжҲ–iOS Appзҡ„е…ій”®пјӣ
  • и®ҫзҪ®жһ¶жһ„armv6 armv7(зңҹжңә)жҲ–i386(жЁЎжӢҹеҷЁ)гҖӮ

жҖ»зҡ„жқҘиҜҙжҳҜжҢәз®ҖеҚ•жҳ“жҮӮзҡ„дёҖе°Ҹж®өд»Јз Ғ~еҸҰеӨ–еҸҜд»ҘеҸӮиҖғStackOverflowдёҠзҡ„How to set up CMake to build an app for the iPhoneпјҢеӣ дёәеңЁios-cmakeйҮҢжҸҗдҫӣзҡ„дҫӢеӯҗеҸӘжңүlibе·ҘзЁӢжІЎжңүappе·ҘзЁӢгҖӮ

иҝҷйҮҢжҲ‘йҖүжӢ©жіЁйҮҠжҺүејәиЎҢи®ҫзҪ®зј–иҜ‘еҷЁйғЁеҲҶпјҢжү“ејҖи·іиҝҮзј–иҜ‘еҷЁжЈҖжҹҘзҡ„д»Јз ҒпјӣдёҚз„¶зҡ„иҜқз”ҹжҲҗзҡ„ж–Үд»¶еҗҚжҳҜgcc42иҖҢйқһclang60~
